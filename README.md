# Farmer Market Intelligence Platform 🌾

## 📌 Project Overview
This project aims to help farmers and traders analyze and predict market prices of crops using Machine Learning.

The system uses historical market data to forecast future prices and support better decision-making.

## 🚧 Project Status
Currently under development

## ✅ Completed
- Data collection (APMC market data)
- Data cleaning and preprocessing
- Exploratory Data Analysis (EDA)

## 🔄 In Progress
- Machine Learning model training
- Price prediction logic

## ⏳ Planned
- Frontend integration
- User interface for predictions
- Deployment

## 🛠️ Tech Stack
- Python
- Pandas
- Scikit-learn
- Jupyter Notebook
- HTML, CSS, JavaScript

## 📊 Features
- Market price trend analysis
- Weekly price prediction
- Data visualization

## 🎯 Future Scope

- Mobile/web application for farmers
